

# bHere@UCF

This is a reimagining of UCF Here where instead of scanning a QR code, students will listen
for a bluetooth advertisement from the teacher. This approach ensures students cannot fake attendance since the attendance code is only accessible by devices listening over bluetooth. 

# This Repository

This repository is for the web version of bHere@UCF. The web version does not have any attendance functionality but does have the following:

- Login
- Register
- View enrolled classes as a student
- View classes you teach as an instructor
- Creating clases as an instructor for students to join
- Joining a class as a student
- Leaving/Deleting classes
- Viewing attendance records for previous class sessions
